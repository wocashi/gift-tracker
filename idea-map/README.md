# IdeaMap — 思考整理ツール

アイデアを投げ込むとAIが自動でクラスタリングして2Dマップを生成します。

## セットアップ

1. `.env.local` ファイルを作成してAnthropicのAPIキーを設定

```bash
cp .env.local.example .env.local
# .env.local を編集して ANTHROPIC_API_KEY を設定
```

2. 開発サーバーを起動

```bash
npm run dev
```

3. ブラウザで http://localhost:3000 を開く

## 使い方

1. 左サイドバーにアイデアを入力（Enterキーで追加）
2. アイデアが2件以上になったら「✨ マップを生成」を押す
3. AIが意味的に近いアイデアをグループ化してマップを生成
4. マップはスクロールでズーム、ドラッグでパン操作
5. 点にホバーするとアイデアの内容を確認
6. 左下のクラスター一覧をクリックすると要約を表示
